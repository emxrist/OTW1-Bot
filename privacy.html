Privacy Policy – OTW1 Bot

Last updated: July 23, 2025

This Privacy Policy outlines what data is (and is not) collected or processed by OTW1 Bot on Discord.

1. Data Collected

OTW1 Bot only processes the following information, temporarily and as needed for functionality:

  • User IDs and message content (during slash command execution only)
  • Guild (server) IDs, channel IDs, and roles (for command logic)
  • Match statistics (e.g., for `/status`) stored temporarily in memory or database

2. Data Storage

No persistent user data is stored outside the Discord platform unless strictly required for bot features. If any storage is used, it is minimal and used exclusively for features like tracking stats or matches.

3. Data Sharing

No user data is ever sold, shared, or transmitted to third parties. OTW1 Bot operates solely for Discord functionality and no analytics or advertising tools are integrated.

4. User Control

If you wish to remove your data or have concerns, you may contact the developer directly via Discord. You may also remove the bot from your server at any time to cease data processing.

5. Security

We take reasonable measures to secure any data handled by the bot and prevent unauthorized access. However, the nature of Discord bots means some data passes through Discord’s servers, which are governed by Discord’s Privacy Policy.

6. Changes

This Privacy Policy may be updated at any time. Continued use of OTW1 Bot constitutes acceptance of any changes.

If you have questions, please contact the bot developer via Discord.
